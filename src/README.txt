# RutaSmart - Backend Distribuido

Proyecto final de Sistemas Distribuidos (Redes III).

## Características
* Backend Spring Boot
* Integración con NFS (Sistema de Archivos Distribuido)
* Criptografía Asimétrica RSA
* Simulación GPS en tiempo real
* Panel Administrativo

## Despliegue
1. Montar NFS en `/mnt/eData`
2. Ejecutar `mvn spring-boot:run`
